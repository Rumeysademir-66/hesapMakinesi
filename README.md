# Calculator App

## Overview

Calculator App is a desktop application developed with C# and Windows Forms. The project provides basic calculator operations through a simple and user-friendly interface.

The purpose of this project is to practice C# programming, Windows Forms application development, event handling, and basic user interface design.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear input function
- Simple and understandable interface

## Technologies Used

- C#
- .NET Framework
- Windows Forms
- Visual Studio
